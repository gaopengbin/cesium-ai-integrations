## Highlights

**Cesium advantages:** richest routing (multi-mode, traffic, turn-by-turn), POI/nearby search, multi-provider flexibility, caching + rate limiting, Zod output validation

**Notable gaps:** no reverse geocoding (all others), no Haversine distance (tpp6me, MCP-Geo), no Docker (yuiseki)

**Notes:**

- **Grounding Lite** — experimental/pre-GA Google Maps MCP server; uniquely offers weather forecasts; explicitly omits step-by-step directions and real-time traffic; requires a Google Cloud project with billing enabled and an API key or OAuth credentials; imposes strict attribution requirements on LLM responses; currently free during experimental phase
- **TomTom MCP** — the most feature-complete navigation-focused MCP in this comparison; available as a hosted remote server (public preview) or local install via npm/Docker; two backends — TomTom Maps (default, stable) and TomTom Orbis Maps (preview, adds EV routing, search-along-route, area search, EV charging search, interactive map widgets, and GeoJSON data visualization); uniquely offers reachable range / isochrone computation and server-side map rendering (static image tile stitching via skia-canvas); interactive maps render directly inside the AI client when using the Orbis backend; requires a TomTom API key (free tier available on developer portal); Apache 2.0 license
